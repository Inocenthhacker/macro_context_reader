## Obiectiv

Livrează un pipeline reproductibil pentru ingestia, validarea și calculul de signals din CFTC COT (Commitment of Traders) Traders in Financial Futures, restrâns la EUR futures contractul standard `EURO FX - CHICAGO MERCANTILE EXCHANGE`. Asigură că `lev_net`, `am_net`, `lev_delta_wow` și `lev_percentile_52w` reflectă o singură serie continuă de date reale, fără contaminare multi-contract.

**Context istoric:** Codul existent în `src/macro_context_reader/positioning/cot_structural.py` (74 linii, 7 teste unit verzi) a fost identificat în PRD-400-INSPECT/CC-1 ca având un bug critic de filtrare: `str.contains("EURO FX", case=False)` matchează 3 contracte CFTC distincte simultan, contaminând Parquet-ul output cu duplicate (787 rânduri pentru 326 date unice). Parquet-ul corupt a fost șters preventiv în commit f8f58dc (CLEANUP-CORRUPT-DATA/CC-1). PRD-400 e responsabil pentru bugfix sistemic + rigorizare metodologică retroactivă conform LESSONS-001.

## Abordare Tehnică

**Sursă date:** CFTC COT Traders in Financial Futures (TFF), Futures-only, via `cot_reports.cot_year(year, traders_in_financial_futures_fut)`.

**Fereastră temporală:** 2018-01-01 → present. TFF dataset începe în 2006, dar pentru EUR futures liquid începe practic în 2018 (validat empiric în diagnostic).

**Filter strategy:** Exact match pe câmpul `Market_and_Exchange_Names == "EURO FX - CHICAGO MERCANTILE EXCHANGE"` (DEC-007 va documenta justificarea).

**Coloane output Parquet:**

| Coloana | Tip | Sursă |
|---|---|---|
| `date` | datetime64[ns] | `As_of_Date_In_Form_YYMMDD` parsed |
| `lev_net` | int64 | `Lev_Money_Positions_Long_All - Lev_Money_Positions_Short_All` |
| `am_net` | int64 | `Asset_Mgr_Positions_Long_All - Asset_Mgr_Positions_Short_All` |
| `lev_delta_wow` | float64 | `lev_net.diff(1)` |
| `lev_percentile_52w` | float64 | `lev_net.rolling(52).rank(pct=True)` |

**Validare layer:** Pydantic `COTStructuralRow` per rând prin `_validate_rows()`, raise `ValidationError` la primul rând invalid.

**Storage:** Parquet snappy compression la `data/positioning/cot_eur.parquet`.

**Code structure:**

```
src/macro_context_reader/positioning/
├── __init__.py
├── cot_structural.py     ← REFACTORED (CC-1)
└── schemas.py            ← NEW (CC-1)

```
