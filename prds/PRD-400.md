## 1. Obiectiv

Construiește pipeline-ul pentru semnalul de poziționare structurală pe EUR/USD folosind două surse CFTC complementare, conform metodologiei Larry Williams ("Trade Stocks and Commodities with the Insiders", Wiley 2005):

1. **Semnal primary (Commercials, follow)** — Legacy COT Futures-only pentru `EURO FX - CHICAGO MERCANTILE EXCHANGE`. Williams COT Index 26-week min-max scaling, praguri 80/20. Interpretare: index > 80 = bullish EUR (commercials buying), index < 20 = bearish EUR.

2. **Semnal secondary (Leveraged Funds, contrarian)** — TFF Futures-only pentru același market. Aceeași formulă Williams 26-week, dar cu interpretare opusă conform Williams page 77: index > 80 = bearish EUR (large speculators crowded long, reversal risk), index < 20 = bullish EUR.

Output-ul ambelor semnale alimentează PRD-300 (Divergence Signal) și PRD-500 (Output Aggregation).

- **REQ-1:** Fetch Legacy COT Futures-only (`legacy_fut`) pentru contractul exact `EURO FX - CHICAGO MERCANTILE EXCHANGE` cu start_year=1999 hardcodat ca floor.
- **REQ-2:** Extrage coloanele `Commercial Positions-Long (All)`, `Commercial Positions-Short (All)`, calculează `comm_net = long − short`.
